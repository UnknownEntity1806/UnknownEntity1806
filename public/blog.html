Posted on 11.01.2026

Work in Progress Jan '26

A little bit of my works in progress in January. Some might be finished this month. Others are more ideas that are for further in the future.

First, I am constantly working and thinking about upgrading the website. My idea for version 2.0 is to have the different portfolio categories have their own themed layout. So I have made some sketches for that. Although I will not start actually working on coding version 2.0 until all the pages are up and running in version 1.2.

Second, I am working on expanding the amount of detail in my drawing from the *Walk in the Forest* universe I made this past November. I really like the colours and everything, but I always seem to give up once I have made 1-2 fairies. I want to try and improve the details in my bigger drawings, so that they become illustrations you can really stare at for a longer time. It's hard to see now, but up until now I have included one other fairy and some falling leaves.

Third, I am working on making some more buttons to put on the *ABOUT* page, which will probably also be available on the *FOR YOU* as a freebie. I know that there are a lot of ready made buttons available already. But I like to try and make things for myself ^^

So, those are the things I am working on this month. As I am also in the process of moving over to the winter-house, it's not a lot. But I wanted to give this update in any case.

Master list of resources and bookmarks

This is a list that I keep updated with useful resources and interesting websites. I share it here because it might also be of use to you ^^

Website making-related

  • Fairytale's Make a Website Guide :: a nice guide to making a website on the indieweb.
  • Porkbun :: a cheap static hosting service.
  • Github :: a code repository. I use it integrated with Porkbun.
  • W3Schools :: a free online tutorial site.
  • Visual Studio Code :: a free code editor for writing your HTML, CSS and Javascript. You can preview your edits. It has extensions to automatically unify the formatting of your code. Insert boilerplate, etc.
  • Udemy Coding Course :: This is the course I personally use to learn about front-end (and back-end) development.
  • Lorem Picsum :: a free placeholder image service.
  • Google Fonts :: a free font service.
  • Free cultural heritage images :: you can download these and use them for content creation.
  • CollectUI :: a collection of UI design resources. (not really in the indieweb style, but still)
  • Interactive Flexbox Tutorial :: a tutorial on CSS flexbox.

other's resource pages

  • Kaylee Rowena's Resource Page :: a great collection of resources for artists and creatives.
